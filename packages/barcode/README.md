# @react-native-camera/barcode

Barcode reader plugin

## Installation

```sh
npm install @react-native-camera/barcode
```

## Usage

```js
import Barcode from "@react-native-camera/barcode";

// ...

const result = await Barcode.multiply(3, 7);
```
